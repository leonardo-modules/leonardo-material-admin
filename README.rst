
=============================
Leonardo Admin Material Theme
=============================

Compatibility layer beetween ``django-material`` and Leonardo CMS / FeinCMS admin.

.. contents::
    :local:

Installation
------------

.. code-block:: bash

    pip install leonardo-material-admin

Disable admin
-------------

.. code-block:: bash

    pip uninstall leonardo-material-admin
